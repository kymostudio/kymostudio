## Highlights

### 1. RealtimeBoard becomes Miro

On **6 March 2019**, RealtimeBoard — then 2M+ users — **rebranded to Miro**, repositioning from "online whiteboarding" to end-to-end visual collaboration. The rebrand marks the infinite collaborative whiteboard's arrival as a category in its own right; one year later, pandemic remote work made the shared canvas indispensable and Miro grew explosively.

### 2. Kroki launches

Yuzu Tech's free, MIT-licensed service (repo created January 2019) exposed **one unified HTTP API over 25+ text-to-diagram engines** (PlantUML, Mermaid, Graphviz, BPMN, C4, Excalidraw…), making diagrams-as-code embeddable in any wiki/CI pipeline and self-hostable. Aggregation as infrastructure: one endpoint, every grammar.

### 3. Mermaid wins the JS Open Source Award

Mermaid took "The most exciting use of technology" at the 2019 JavaScript Open Source Awards — the moment the project graduated from niche markdown plugin to recognized mainstream infrastructure, three years before GitHub adopted it.
