## Highlights

### 1. The AI text-to-diagram wave

On **18 May 2023** Eraser launched **DiagramGPT** — flowcharts, ERDs, cloud-architecture and sequence diagrams generated from natural language or code via GPT-4 (it drove ~30% of Eraser sign-ups). In **November 2023** tldraw's viral **"Make Real"** demo used GPT-4V to turn hand-drawn wireframes into working UI code. 2023 flipped the authoring direction: the LLM writes the diagram source from intent — and the AI tools converged on *emitting diagram-as-code formats*, vindicating text-based diagram languages as the natural LLM-to-picture interface.

### 2. Mermaid v10 ships as ESM-only

Released 21 February 2023: a breaking modernization — ESM-only packaging, async render API, lazy-loaded diagram types — that made Mermaid embeddable in the modern bundler ecosystem just as every notes app, wiki, and AI chatbot was adopting it.

### 3. Figma launches Dev Mode at Config

Announced 21 June 2023: a developer-facing inspector workspace bridging design artifacts and code (VS Code plugin, tokens, code snippets) — the design-to-implementation handoff becoming a first-class product, and Figma's biggest platform expansion of the year.

## Context

ChatGPT plugins, Mermaid-emitting chatbots, and AI features in Miro/FigJam all landed the same year. The pattern that stuck: the LLM generates the *source*, a deterministic renderer draws it — hallucination-resistant because the renderer enforces the grammar. This is kymo's strongest tailwind: a strict, specified DSL is exactly what an LLM should emit.
