## Highlights

### 1. Visio 2007 — diagrams become data-driven

**Microsoft Visio 2007** made its flagship pitch **data-linked diagrams**: shapes bound to external data (Excel, SQL, SharePoint) rendered as **Data Graphics** that refresh as the source changes. The diagram became a live view of data, not a static picture — anticipating the diagram-as-code principle that a diagram should be *derived* from data, not maintained by hand.

### 2. MindMeister launches collaborative web mind mapping

MeisterLabs (Munich) launched **MindMeister** (beta February, public May 2007) — the first widely adopted real-time collaborative mind-mapping web app, "Google Docs for mind maps." It extended the web-app diagramming model beyond flowcharts into ideation diagrams and eventually grew past 20M users.

### 3. XMind 1.0 released

Founded in Shenzhen in 2006, XMind shipped its first official release in April 2007 — an Eclipse-RCP desktop mind-mapping tool that won "Best Commercial RCP Application" at EclipseCon 2008. With MindMeister, it marks 2007 as the breakout year of the mind-mapping software wave (and set up XMind's influential open-sourcing in 2008).

## Context

2007 is thin on landmark text-to-diagram events — WebSequenceDiagrams (2008) and PlantUML (2009) were just ahead. The durable themes are Visio's data-binding ("change the data, not the drawing" — literally kymo's architecture) and the mind-mapping wave broadening who diagrams.
