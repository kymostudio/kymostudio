## Highlights

### 1. Excalidraw is born; draw.io becomes diagrams.net

On **1 January 2020**, Christopher "vjeux" Chedeau started **Excalidraw** as a holiday side project: an open-source whiteboard with a deliberately **hand-drawn aesthetic**. Propelled by pandemic remote work it became a full collaborative product within a year (~20k weekly actives by January 2021) — the developer-favorite open-source canvas and a reference implementation countless tools embed. The same year **draw.io announced its rebrand to diagrams.net** (February 2020).

### 2. `diagrams` (mingrammer) launches — cloud architecture as Python code

Released February 2020, the "Diagram as Code" Python library rendered AWS/GCP/Azure/K8s architecture diagrams from a few lines of Python and went viral immediately (42k+ stars today) — crystallizing the cloud-architecture-diagram boom into a code-first workflow.

### 3. Miro raises $50M Series B as remote work explodes

Closed April 2020, led by ICONIQ, weeks into COVID lockdowns; Miro had 5M users, 20,000 paying customers and 80% of the Fortune 100. The funding event that marks whiteboard diagramming's pandemic inflection.

## Context

Excalidraw's sketchy style made a statement — rough-looking diagrams invite discussion, polished ones discourage it — and its architecture mattered too: local-first, end-to-end-encrypted collaboration, fully open source. kymo ships an Excalidraw output target precisely because its open format became a lingua franca for editable sketches.
