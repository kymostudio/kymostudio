## Highlights

### 1. Diagrams become agent-operable: canvas-as-SDK and MCP

On **9 April 2025** **tldraw raised a $10M Series A** (Lux Capital, Definition) to build the infinite canvas as an **embeddable developer SDK** — already powering ClickUp Whiteboards, Padlet, and others. In parallel, diagramming converged with AI agents via the **Model Context Protocol**: official and community MCP servers for Mermaid Chart and Excalidraw let Claude and other agents create, inspect, and iteratively edit live diagrams inside coding workflows. The 1995 question was "how should humans draw software?"; the 2025 question is "how do humans *and agents* share a visual model?"

### 2. Figma IPOs on the NYSE

On **31 July 2025** Figma priced at $33, opened at $85, and closed day one at $115.50 (+250%) for a ~$68B valuation — the largest liquidity event ever for a visual-collaboration company, and vindication after the collapsed $20B Adobe deal.

### 3. Figma Make (and Sites/Draw/Buzz) launch at Config

Announced 7 May 2025: Figma's prompt-to-app AI tool turns design files or plain-language descriptions into working prototypes and web apps (built on Claude), part of a five-product Config blitz pushing Figma from design canvas to full visual-creation platform.

## Context

This thread is still unfolding: agentic coding tools generate architecture diagrams as part of normal work, and the diagram formats that win are the ones agents can emit and validate — the diagram-as-code lineage this timeline has traced since Graphviz. kymo's MCP server, strict DSL, and deterministic animated-SVG renderer are built for exactly this inflection.
