## Highlights

### 1. SVG 1.0 becomes a W3C Recommendation

On **4 September 2001**, **SVG 1.0 was published as a full W3C Recommendation** — promoted from draft to officially ratified web standard. Recommendation status locked down the shape, path, styling, and animation model that diagrams would later be rendered with everywhere; by the 2010s, essentially every web diagramming tool emitted SVG.

### 2. OmniGraffle 1.0 ships for Mac OS X

The Omni Group's diagrammer launched 7 May 2001 alongside the new OS X platform and quickly became *the* Mac diagramming standard in the post-acquisition vacuum (Visio's Mac story was dead) — seeding the design-friendly aesthetic that later web tools imitated.

### 3. OMG launches Model Driven Architecture (MDA)

Announced early 2001 and voted the OMG's base architecture in September 2001, **MDA** promoted diagrams/models from documentation to *primary development artifacts* — driving the decade's CASE-tool renaissance (Rational XDE, Enterprise Architect, Together) and the UML 2.0 work.

## Context

Native browser SVG support still lagged for years — early SVG often needed Adobe's plugin viewer — so desktop tools kept the authoring lead through the 2000s. But the standard was now real, and the modular SVG 1.1 followed in 2003.
