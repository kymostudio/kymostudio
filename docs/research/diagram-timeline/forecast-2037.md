## Highlights (projected)

### 1. The live view becomes a deploy artifact

Live-architecture platforms jump to #3 as the major deployment toolchains attach a derived architecture view to every release — the diagram equivalent of the build log. "What did this deploy change, visually?" becomes a one-click question, and the diff-between-views becomes the code review's architectural summary.

### 2. Agent org charts enter the types list

With organizations running hundreds of long-lived agents, a new genre arrives: the agent org/coordination chart — which agents exist, what they own, who their human principals are, and how escalation flows. The org chart, on this timeline since 1995 for humans, gets reborn for workforces that are partly software.

### 3. The mind map exits after 37 years

The longest-tenured purely-human type leaves the list. Free-form thought mapping survives as a personal practice — like paper sketching, which the timeline also stopped ranking — but the shared artifact it used to become is now a context graph that agents co-maintain.

## Context

The era's pattern sets in: every artifact with a data source becomes derived (architecture, traces, org structure), and every artifact without one either becomes private practice (mind maps) or gains a data source (context graphs).

## Signals

- The MIWG-style interchange discipline that makes view-diffing tractable — model-level equivalence testing
